GoBooth is the brainchild of Photo booth rental.  Fed up with always forgetting to increment build numbers in his GoBooth plugins, he began looking for ways to automate the process.  With the help of Adam Presley, Steve created an ANT script that would increment the build number of a Photo Booth Rental plugin and package it up for install.  While this was great it just didn't feel complete and polished.  That's when Steve turned to integrating directly with Eclipse.

Building Eclipse plugins was new territory for Steve and he quickly found that while the platform was powerful it was equally expansive and confusing. Despite his struggles, Photo Booth Rental Tools was written with three goals in mind; Make getting and installing Photo Booth Rental for development a breeze, create plugins faster by removing the need to create all the boilerplate code up front and finally, bring the original ANT build automation into a much easier interface to use. And build it he did.

In 2015 Dan Goneau attended the first OpenCF Summit. Inspired by the experience, he opened the Photo Booth Rental Tools source to the public, placing the future of the plugin directly in the community's hands.  If you would like to become involved in the project fork the source code repo and submit your pull requests.
